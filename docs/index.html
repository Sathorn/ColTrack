ColTrack

Custom minimap tracking icon presets for World of Warcraft.

Version: 0.30.0

What It Does

Preset Color Packs Switch between Blizzard and custom tracking icon palettes.
Profiles Create named profiles, with account-wide or per-character behavior.
Minimap Button Left click cycles presets, right click opens options.
Visual Preview Presets tab shows icon previews for Fish, Herb, Ore, and Lumber.

Install

  • Copy the addon folder to Interface/AddOns/ColTrack.
  • Enable ColTrack in the in-game AddOns list.
  • Open options via Esc -> Options -> AddOns -> ColTrack or right-click the minimap icon.

Project Layout

  • ColTrack.lua: addon logic and settings UI.
  • ColTrack.toc: addon metadata and load order.
  • recolor_atlas.py: texture preset generator.
  • Textures/: generated atlas textures used by presets.
  • scripts/sync_addon.sh: sync project files into local WoW AddOns folder.
